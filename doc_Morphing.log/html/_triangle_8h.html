Detailed Description

Gère la construction et l'utilisation de la structure Triangle.

  1. Structure Triangle : permet de représenter un triangle avec ses trois sommets.
    Contient 3 strcutures de type point.
  2. Structure CelTriangle/ListeTriangle : permet de représenter une liste de triangle ou une cellule.
    Contient un triangle, 3 pointeurs vers les 3 possibles voisins de ce triangle, contient aussi un pointeur vers la paire du triangle dans l'image de droite. le pointeur suivant correspond au triangle suivant dans la liste.
