Detailed Description

Gère l'affichage du projet.

  1. Structure Bouton : permet de représenter un bouton.
    Contient son texte et ses coordonées.
  2. Structure Interface : permet de gerer l'interface.
    Contient les 2 images, leurs coordonées et les boutons.
